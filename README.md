# Medimint Protocol (MDMT)

Medimint is a blockchain-powered platform that tokenizes medical infrastructure and enables secure, transparent access to healthcare services.

## 🔗 Live Contract
- Network: Ethereum Mainnet
- Contract Address: 0x7d697c675f9578e986f6af4260dcd47ec3546b38
- Token: MDMT
- Supply: 1,000,000,000 (Fixed)

## 📄 Documentation
https://avisa-1.gitbook.io/medimint

## ⚙️ Smart Contract
The MDMT token is a fixed-supply ERC-20 token deployed on Ethereum.

## 🚀 Vision
Medimint connects real-world healthcare infrastructure with blockchain-based financial systems to create a scalable and transparent healthcare ecosystem.
